## 🎈介绍
优雅，功能强大，易于阅读的Gridea主题。

## 🌏示例
- [示例站点](https://blog.itjoker.cn)
- [我的博客](https://blog.itjoker.cn)

## 💡特征
- 适当的空白块，优雅但不简单。

- 浅色/深色主题，只需单击一下。

- 丰富的高亮模式。

- 精心挑选的字体，最佳阅读体验。 \ * *特别推荐使用“ Microsoft Jhenghei”。

- 自动适应移动设备和屏幕。

- 支持中英文切换

- 支持MathJax，支持用LaTeX编写的公式。

- 支持Live 2d

- 友好的SEO

$$f(x) = a_1x^n + a_2x^{n-1} + a_3x^{n-2}$$

```bash
cd your-blog/themes

```
